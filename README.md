CALCULATOR PROJECT :
A simple and responsive calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and also supports keyboard input.
🚀 Features
Basic operations: +, -, *, /
Clean and modern UI
Responsive design
Keyboard support
Error handling for invalid inputs
Clear (C) button to reset display

🛠️ Technologies Used
HTML5
CSS3 (Flexbox + Grid)
JavaScript (DOM manipulation)

🎮 How to Use
Open the index.html file in your browser
Click buttons to perform calculations
Or use your keyboard
